## üöÄ Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
- Modern web browser

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/rm-groups.git

# Navigate to project directory
cd rm-groups

# Install dependencies
npm install

# Start development server
npm run dev
```

### Build for Production
```bash
# Build the application
npm run build

# Preview production build
npm run preview
```

## üîß Configuration

### Environment Variables
Create a `.env.local` file:
```env
GEMINI_API_KEY=your_gemini_api_key_here
```

### Google Analytics Setup
1. Replace `G-RMGROUPS2026` with your GA4 tracking ID in:
   - `index.html`
   - `components/GoogleAnalytics.tsx`
2. Replace `GTM-RMGROUPS` with your GTM container ID in `index.html`

### PWA Configuration
Update `public/manifest.json` with your app details:
- App name and description
- Icons and screenshots
- Theme colors
- Start URL
